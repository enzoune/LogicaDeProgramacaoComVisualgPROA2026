> "A sintaxe é passageira, a lógica é eterna."

Este repositório não é apenas uma coleção de arquivos `.alg`. Ele representa a construção da minha fundação técnica como Desenvolvedor Full-Stack. Aqui, foco no que realmente importa antes de qualquer framework: a capacidade de decompor problemas complexos em **Algoritmos Estruturados**.

## 🚀 Por que a Lógica precede a Sintaxe?

Muitos iniciantes pulam diretamente para bibliotecas e frameworks modernos. Minha abordagem foi diferente: utilizei as metodologias de **Manzano** e **Faccat** para dominar o **Fluxo de Controle** e a manipulação de dados em sua forma mais pura (Portugol). 

Ao dominar essas estruturas no **VisualG**, garanto que minha transição para linguagens como JavaScript ou Python seja baseada em entendimento arquitetural, e não apenas em memorização de comandos.

## 📚 Conteúdo do Repositório

Os exercícios aqui contidos abordam os pilares fundamentais da computação:

| Categoria | Descrição | Conceitos Chave |
| :--- | :--- | :--- |
| **Operações Básicas** | Cálculos matemáticos e conversões | Entrada/Saída, Operadores |

| **Laços de Repetição** | Automação de tarefas iterativas | Enquanto, Repita, Para |

## 🧩 Resolução de Problemas

Cada algoritmo neste repositório segue os princípios da **Escrita Técnica Limpa**:
1.  **Declaração de Variáveis**: Nomeação clara e tipagem correta.
2.  **Identação**: Código estruturado para facilitar a leitura.
3.  **Eficiência**: Foco em encontrar a solução mais lógica para o desafio proposto.
